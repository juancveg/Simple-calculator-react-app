# 🧮 Simple Calculator in React

A simple calculator application built with React, similar to the basic iOS/macOS calculator. It allows for basic arithmetic operations with a clean and responsive interface.

## ✨ Features

- Addition, subtraction, multiplication, and division
- Sign toggle (+/-)
- Percentage calculation (%)
- Decimal point
- Clear button (AC)
- Error handling (e.g., division by zero)

## 🛠️ Technologies

- React 19
- Create React App (react-scripts 5)
- CSS

## 📂 Project Structure

```
src/
├── components/
│   ├── Calculator.js   # Calculator logic and UI
│   └── Calculator.css  # Calculator styles
├── App.js
└── index.js
```

## 🚀 Installation and Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   cd Simple-calculator-react-app-master
   ```

2. Install the dependencies:
   ```
   npm install
   ```

3. Start the application in development mode:
   ```
   npm start
   ```

4. Open http://localhost:3000 in your browser.

## 📜 Available Scripts

- `npm start` – Runs the app in development mode.
- `npm run build` – Builds the app for production in the build folder.
- `npm test` – Launches the test runner.

## 📌 Notes

This project was created with Create React App as a baseline to practice state management in React (useState) through the development of a functional calculator.
